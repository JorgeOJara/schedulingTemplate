# Scheduling API

Backend API for the scheduling application.

## Development

```bash
# Install dependencies
bun install

# Copy environment file
cp .env.example .env

# Update .env with your values

# Start development server
bun run dev

# Run migrations
bun prisma migrate dev

# Generate Prisma client
bun prisma generate

# Open Prisma Studio
bun prisma studio

# Build for production
tsc --build
```
